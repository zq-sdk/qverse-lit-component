# @qverse-ui/lit-components

## 安装

```bash
npm install @qverse-ui/lit-components
```

## 使用

### 全部组件

注册 `base` + `common` 全部 Web Component：

```ts
import '@qverse-ui/lit-components'
```

### 基础组件

仅注册 `lit-button`：

```ts
import '@qverse-ui/lit-components/base'
```

### 通用组件

仅注册视图切换组件（`lit-switch-panorama-view`、`lit-switch-dollhouse-view`、`lit-switch-floorplan-view`）：

```ts
import '@qverse-ui/lit-components/common'
```
